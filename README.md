# Laravel Docker Templete

## Generate New Laravel Project 

This repository provides a boilerplate for setting up a Laravel project with Docker.  

### Dependency Versions  

- **Composer**: `2.6.5`  
- **APINE**: `3.20`  
- **PHP**: `8.3`  
- **MariaDB**: `10.6`
- **PhpMyAdmin**: `latest`  
- **Redis**: `7.2`  
- **Node.js**: `18-apline`  
- **Mailhog**: `latest`
